# 🌿 ÉcoWatt — Intelligence Énergétique Domestique

ÉcoWatt est une plateforme analytique de pointe conçue pour transformer la gestion de votre consommation électrique. Alliant design premium et précision algorithmique, elle permet d'identifier les leviers d'économies réels de votre foyer.

![Tech Stack](https://img.shields.io/badge/stack-React_19_|_TanStack_|_Tailwind_4-61DAFB.svg)
![Status](https://img.shields.io/badge/status-production_ready-success.svg)

## Fonctionnalités

### 📊 Pilotage Stratégique
- **Dashboard Holistique** : Visualisation granulaire de la consommation via des graphiques interactifs haute fidélité (Chart.js).
- **Calculateur de Précision** : Moteur de calcul adaptatif supportant les tarifs réglementés (EDF) et les offres de marché.

### 🔌 Gestion Intelligente du Parc
- **Audit d'Appareils** : Inventaire complet avec calcul automatique des coûts mensuels par unité.
- **Bibliothèque d'Appareils Types** : Base de données préconfigurée pour une mise en service rapide.

### 💡 Centre d'Optimisation
- **Conseils Contextuels** : Plus de 20 actions d'économies classées par impact (facile, moyen, effort).
- **Scénarios de Réduction** : Simulations d'économies ciblées (5% à 30%).

## Technologies utilisées

| Couche | Technologie |
|--------|-------------|
| **Framework** | TanStack Start (Full-stack React 19) |
| **Routing** | TanStack Router (Type-safe routing) |
| **Build** | Vite 7 |
| **Styling** | Tailwind CSS 4 (Architecture CSS-next) |
| **Charts** | Chart.js / react-chartjs-2 |
| **Icons** | Lucide React |
| **Privacy** | localStorage (Zéro serveur, confidentialité totale) |
| **Host** | Netlify Edge |

## 🚀 Installation & Développement

### Prérequis
- Node.js >= 22.0.0

### Setup

```bash
npm install
```

L'application sera disponible sur [http://localhost:3000](http://localhost:3000).

## Construction pour la production

```bash
npm run build
```
